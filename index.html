BusinessConnect combines the best of professional networking, real-time messaging, and next-generation channel-based communication — all inside a clean, premium and beautifully designed UI.

Nhost — Auth & Profiles Supabase — Messaging & Groups ImageKit — Media & Uploads
Real-Time Messaging at Its Best
Powered by Supabase for real-time messaging
  • Lightning-fast message delivery & zero-delay sync
  • Voice messages with waveform player
  • Images, files, audio & documents via ImageKit
WhatsApp-Style Chats
Powered by Supabase for real-time messaging
  • Private one-to-one encrypted-style chats
  • Message request system & unread badges
  • Seen status & online/offline indicators
Telegram-Style Channels
Powered by Supabase for real-time messaging
  • Public Channels — Anyone can join, unlimited members, rich media
  • Private Channels — Invite-only, hidden from search, admin-only posting
  • Broadcast Channels — Admin posts only, followers react/view
  • Admin controls: add/remove members, pin messages, manage privacy
  • Clean feed, member lists, join/leave buttons, smooth animations
Professional Dashboard & Home Feed
Powered by Nhost for profile & dashboard data
  • Live feed stats — network connection count
  • Business card-style user cards with avatar, job title, company & bio
  • Private accounts with lock badge & message request system
  • Profile photo, username, bio & unique UID with tap-to-copy
  • Saved accounts & instant multi-account switch with manage accounts modal
Smart Search System
Powered by Nhost for user profile search & discovery
  • Live user search with instant auto suggestions
  • Search by name, username, email or UID
  • User avatars, status & bio preview in results
  • Clean dropdown result cards with WhatsApp-style search box
Dark Mode & Settings
Powered by Nhost for user preferences & settings storage
  • Full dark theme — sidebar, chats, modals & feed cards
  • Smooth color transitions & premium contrast
  • Profile, privacy, notification & appearance settings
  • Scalable sidebar, keyboard-friendly & smooth modals
Performance & Scalability
Powered by ImageKit for CDN media delivery & optimization
  • Fast loading, smooth animations & clean DOM
  • CDN-powered media & cloud-secured data
  • Optimized for low network conditions
  • Memory-friendly chat rendering
